# BTCPay Server のデザイン

# なぜデザインが重要か

BTCPay Server のデザインは、エンドユーザーの全体的な体験を改善し、メニューやオプションなどをより分かりやすくすることです。

BTCPay Server のデザインに貢献するには、現在の議論は UX/UI channel (opens new window) を確認し、進行中のデザインプロジェクトは Figma の getting started (opens new window) セクションを確認してください。

グラフィックコンテンツも BTCPay Server の普及に役立ちます。フルサイズのロゴ、ミーム、その他のグラフィックなど、コントリビューターが作るものを見るのを私たちは常に楽しみにしています。 グラフィックコンテンツの一貫性を保つため、Codified Design Guidelines (opens new window) に従うことは必須です。
