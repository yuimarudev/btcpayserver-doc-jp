# そのほかの方法で BTCPay Server に貢献する

  • 動画とグラフィックを作成する
  • ホスティングプロバイダー
  • Directory への貢献
  • プロジェクトディスカッション

  • 情報を広める

# 動画とグラフィックを作成する

機能を紹介したり、BTCPay Server の使い方を示す動画を作成することは、非常に有益な貢献です。 動画チュートリアルは、新規ユーザーが BTCPay Server でできることを理解し、操作に慣れる助けにもなります。

参考として、公式 BTCPay Server YouTube channel (opens new window) にある既存動画の例を確認できます。

# ホスティングプロバイダー

BTCPay Server コミュニティは、1-Click BTCPay Server deployment を実装する VPS ホスティングプロバイダーをさらに求めています。これにより、ソフトウェアの導入をより身近にし、分散化と普及を進められます。

# Directory への貢献

Directory は、加盟店チェックアウト、寄付、資金調達、その他ユーザーのさまざまなユースケースにおける BTCPay Server の可能性を紹介するために作成されました。

このソフトウェアを利用している Web サイトの網羅的ではない一覧が含まれています。

誰でも add a person or an organisation (opens new window) から Directory (opens new window) に追加できます。 GitHub の使い方が分かる場合は、.vuepress/list.js を編集して直接プルリクエストでエントリーを追加することもできます。

Directory に受け付けられるのは BTCPay Server を利用している Web サイトのみで、エントリーは英語であることが求められる点に注意してください。

# プロジェクトディスカッション

BTCPay Discussions Page (opens new window) を確認し、新機能のアイデアやその他のトピックにフィードバックを提供してください。アイデアとしては有望だが、まだ開発段階ではない機能はここで扱われます。

# コミュニティ

開発者でなくても BTCPay Server を支援できます。

もっとも簡単なのは、個人または事業としてソフトウェアを使い、フィードバックを提供し、自分または顧客が遭遇した バグや問題を報告 (opens new window) することです。もう一つの有効な方法は community に参加し、BTCPay Server の利用経験から得た情報を共有して、他の人のトラブルシューティングを手助けすることです。

コミュニティがあなたを助けてくれたように、新しく参加する人を助けることも検討してください。

# 情報を広める

BTCPay Server の理念や提供価値が好きなら、ぜひ広めてください。もっとも簡単で効果的な貢献の一つです。
